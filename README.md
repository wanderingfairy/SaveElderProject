# SaveElderProject

패스트 캠퍼스 iOS School에서 진행했던 1차 해커톤 프로젝트 입니다.



## Description

- 기간 : 2020.01.15

- 사용 기술

  - Language : Swift
  - Framework: UIKit, MapKit, CoreLocation
  - Back-End : Google Firebase

- 팀원 : 정의석, 박인준, 이희진 (iOS 3명)

- 맡은 역할 : 프로젝트 기획, 메인 코딩, Firebase를 이용한 데이터베이스 관리, 로그인과 푸시 알림 등 주요 기능 구현.
